Adzanix is a minimal, no-distraction prayer time reminder for Termux/Linux, written in Go with automatic adhan playback.

---

## Features

- Automatic adhan playback
- Custom prayer schedule via config.json
- Timezone support

---

## Requirements

- Go (for building)
- mpv (for audio playback)

---

## Installation

```bash
git clone https://github.com/yourname/adzanix.git
cd adzanix
```

---

## Usage

1. Edit `config.json`:
   ```
   {
    "timezone": "Asia/Jakarta",
    "adzan_normal": "/path/to/adzan.mp3",
    "adzan_subuh": "/path/to/adzan_subuh.mp3",
    "pray_times": {
      "fajr": "04:45",
      "dhuhr": "12:05",
      "asr": "15:20",
      "maghrib": "18:10",
      "isha": "19:20"
     }
   }
   ```

2. Run:
   ```bash
   go run main.go
   ```

---

## Notes

- Make sure `mpv` is installed:
  ```bash
  apt install mpv
  ```
- Run in the background:
  ```bash
  go run main.go &
  ```
